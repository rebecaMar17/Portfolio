## 🛠️ Características Técnicas del Portafolio

Aunque mi hábitat natural es el Backend, he construido esta interfaz para presentar mi trabajo:
* **Vanilla JS + Arquitectura SPA:** Navegación por pestañas sin recargar la página.
* **Integración de API Rest:** Consumo dinámico de la API pública de GitHub para mostrar mis últimos repositorios actualizados en tiempo real.
* **Animaciones Fluídas:** Uso de `GSAP` para transiciones limpias y un efecto de "linterna" reactivo al ratón mediante CSS variables.
* **Responsive & Tema Oscuro:** Diseño fluido centrado en la legibilidad y la experiencia de usuario (UI/UX).
